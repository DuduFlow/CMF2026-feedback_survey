# CMF 2026 旅程滿意度調查

直接開啟 `index.html` 即可預覽。

## 收集回覆

本頁使用 Google 試算表與 Google Apps Script 儲存回覆，不需要 Netlify。設定方式請見 `google-apps-script/設定步驟.md`。

若尚未在 `config.js` 貼上 Apps Script 網址，送出結果只會存入瀏覽器的 `localStorage`，並顯示示範票數與留言；完成設定後才會顯示所有填答者的真實資料。

## 問卷內容

- 姓名選填
- 6 題五顆星滿意度評分
- 1 題最有印象的三堂課（限選三堂）
- 1 題自由回饋（選填）
- 1 題 2027 長沙 CMF 參加意願
- 1 題公開留言板

送出後會公開顯示每一題的統計與全部文字回覆；公開留言不截斷筆數。
